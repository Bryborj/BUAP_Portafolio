#### Proyecto 3: Evaluador de calificaciones.

**Descripción**

El programa pedirá al usuario una calificacion de un dígito, del 0 al 9, y mostrará si el alumno aprobó o reprobó.

**Funcionamiento esperado**

1. Mostrar un mensaje solicitando una calificación.
2. Leer una calificación desde el teclado.
3. Convertir el carácter ASCII a número.
4. Comparar la calificación ingresada.
5. Si la calificación es mayor o igual a 6, mostrar: **"Alumno aprobado".**
6. Si la calificación es menor que 6, mostrar: **"Alumno reprobado"**.
7. Preguntar si desea evaluar otra calificación.
8. SI el usuario presiona S, el programa repite el proceso.
9. Si el usuario presiona N, el programa termina.
